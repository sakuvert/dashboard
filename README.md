# Monitor Operacional · Zona Drop Off

Proyecto académico — Colegio Genios Traviesos, Puerto Montt.

Panel web de una sola página (HTML, CSS y JavaScript) con KPIs, KRIs, gráficos y matriz de riesgos. Los datos son simulados en el navegador; no requiere backend, base de datos ni PHP.

## Ver el dashboard

**En tu PC:** abre `index.html` con doble clic o arrástralo al navegador.

**En GitHub Pages:** Settings → Pages → Branch `main` → `/root` → Save

https://sakuvert.github.io/dashboard/
