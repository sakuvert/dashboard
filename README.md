# dashboard

Monitor operacional · Zona Drop Off · Colegio Genios Traviesos

Dashboard web estático (HTML/CSS/JS) para visualización de KPIs, riesgos y métricas del proyecto.

## Uso

Abrir `index.php` en el navegador, o servir la carpeta con un servidor local:

```bash
php -S localhost:8000
```

Luego visitar http://localhost:8000/index.php
